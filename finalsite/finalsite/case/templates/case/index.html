教師資訊

本中心是由台科大(國立台灣科技大學)的學生所組成,全都是一對一到府上課,師資雖不一定有教學經驗,但富耐心和愛心,且所有教師皆可免費試教一小時,待試教滿意後才正式聘請,定當為您挑選最適任的老師。

家教費用

  1. 1.國小:國小全科(300-500/hr)
  2. 2.國中:國一國二(350-550/hr) 國三(400-600/hr)
  3. 3.高中:高一高二(500-700/hr) 高三(550-750/hr)
  4. 4.其他:大學專科(700-900/hr) 運動及他技藝(500-700/hr)

p.s.費用因教學經驗而有所不同

注意事項

  1. 1.本中心依據電腦個人保護法,個人資料絶對不會外洩
  2. 2.本中心及家長可先檢查教師的學生證及身份證,確認資格
  3. 3.本中心在家長登記後,將在24小時內與您聯絡
  4. 4.本中心全年無休,隨時為家長服務

讀書方法

正確的讀書方法必須包含:課前預習、上課注意聽講、課後複習、作筆記、
讀書計劃、多做題目、思考回想法,能掌握這些技巧的人,相信必是考試的
常勝軍,若是能有家教在孩子遇到不懂的問題能馬上給予回覆,定能事半功倍。
